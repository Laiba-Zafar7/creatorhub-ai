## вҡҷпёҸ Installation

### Prerequisites
- вң… Node.js v16+ installed
- вң… Firebase account
- вң… Git installed

### Steps

**1пёҸвғЈ Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/quizcloud-app.git
cd quizcloud-app
```

**2пёҸвғЈ Install dependencies**
```bash
npm install
```

**3пёҸвғЈ Create environment file and add Firebase keys**
```env
REACT_APP_FIREBASE_API_KEY=your_api_key
REACT_APP_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
REACT_APP_FIREBASE_PROJECT_ID=your_project_id
REACT_APP_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
REACT_APP_FIREBASE_APP_ID=your_app_id
```

**4пёҸвғЈ Start the app**
```bash
npm start
```

рҹҺү App runs at **http://localhost:3000**

## рҹ”Ҙ Firebase Setup

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Create a new project named `QuizCloud`
3. Enable **Email/Password** Authentication
4. Create a **Firestore Database** in test mode
5. Register a Web App and copy config keys to `.env`
6. Add Firestore Security Rules from the project documentation
